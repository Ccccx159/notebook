# 191.位-1-的个数

## 题目描述
编写一个函数，输入是一个无符号整数（以二进制串的形式），返回其二进制表达式中 设置位 的个数（也被称为汉明重量）。

示例 1：

```txt
输入：n = 11
输出：3
解释：输入的二进制串 
1011 中，共有 3 个设置位。
```

示例 2：

```txt
输入：n = 128
输出：1
解释：输入的二进制串 10000000 中，共有 1 个设置位。
```

示例 3：

```txt
输入：n = 2147483645
输出：30
解释：输入的二进制串 11111111111111111111111111111101 中，共有 30 个设置位。
``` 

提示：

+ 1 <= n <= 2^31 - 1

## 题解

### 解法一：逐位计算

```cpp
class Solution {
public:
  int hammingWeight(int n) {
    int cnt = 0, x = 0;
    while (x < 32) {
      if (n & (1 << x++)) cnt++;
    }
    return cnt;
  }
};
```

## 总结

逐位计算即可，注意循环条件为 32 次。
